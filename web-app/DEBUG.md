# 調試步驟（Linus 原則：簡化問題）

## 問題診斷：

### 1. 檢查 Canvas 是否正常載入
- 打開 http://192.168.1.101:3002
- 頁面應該顯示彩色地圖 + 紫藍色遮罩層

### 2. 測試觸控刮開
- 在地圖任意位置滑動手指
- **預期**：紫藍色遮罩被刮開，露出底下彩色地圖
- **如果不行**：Canvas 觸控事件有問題

### 3. 測試點擊雪場
- 點擊側邊欄的雪場列表（如：二世谷）
- **預期**：地圖自動放大到該雪場，顯示底部提示 + 脈衝圈
- **如果不行**：ViewBox 設置或狀態更新有問題

## 快速修復方案：

如果觸控不工作，最可能原因：
1. Canvas 的 `pointer-events` 被設為 none
2. z-index 層級問題導致被其他層覆蓋
3. Touch 事件沒有正確綁定
